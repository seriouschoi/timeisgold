# 타임 슬롯 편집 고도화.

## 슬롯 편집뷰가 표시된 상태에서, 목록 동작 변경.
- 선택된 슬롯만 드래그 가능. 나머지 요소는 드래그 불가.
- 스크롤: 목록 스크롤.

## 타임 슬롯 에딧 뷰의 시간 선택 범위 제약설정하기.
- 슬롯의 전후 시간 까지만 허용.
- 지금 생각해보니 드래그해서 바꾸기보다..이걸 먼저 만들걸 그랬네...
- 엔드 타임이 시작 시간보다 작게 설정 불가.
- 시작 시간이 엔드타임보다 늦게 설정 불가..?
  - 이거.. 자정 넘어가는거 설정하려면 허락 해야 할듯.

자정 넘어가는걸 감안해서 제약해야 하네..

일단.. 이전 슬롯, 이후 슬롯 범위로 전달하고.

아래의 두 타임은 선택할 수 없게 개발한다.
00:00 ~ 이전슬롯
이후 슬롯 ~ 24:00

## 예시:
### 파라미터.
이전: 10:00 
이후: 14:00
### 선택 불가 시간.
00:00 ~ 10:00
14:00 ~ 24:00

### 파라미터.
이전: 21:00 
이후: 05:00
### 선택 불가 시간.
00:00 ~ 21:00
05:00 ~ 24:00

시작 시간을 23시로 설정하고,
종료 시간을.. 21시로 설정 할 일은 없군.

제약이 없을 경우.
시작 시간: 21시.
종료 시간: 05시.
이거 안될 이유는 없구나.
-> 근데 동작이 안되네. 이거 잡아야 할듯.
