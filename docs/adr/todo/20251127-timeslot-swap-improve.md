# 타임 슬롯 스왑 기능의 고도화.

## 문제점.
현재 드래그로 스왑 제공.
허나 이 기능 자체가 그렇게 쓰지 편하지 않음.
* UX가 편하지 않음.
  * 써보니깐, 의도치 않은 드래그로 상단 아이템과 스왑이 될때, 화면의 요소가 드래그를 벗어나서 튀는걸로 느껴짐.(일종의 번쩍거림 처럼 느껴짐)
* 유지보수에도 좋지 않음. 
  * 스왑시, 드래그 중인 아이템외 다른 아이템과 상호작용이 일어나야함.
  * 이 상호작용으로 인해, 드래그중 의도치 않은 상호작용때마다 아이템들의 일괄 변경 요청이 일어나야함.

즉 구현난이도도 높고, 사이드이펙트도 많은데, 쓰기 편하지도 않음.

## 해결책 
차라리 아이템 선택기능을 넣고, 선택한 아이템은 우측에 스왑 버튼을 노출시키자.
좀더 고민해서..사용자가 선택한 아이템만 드래그가 되게 하는건 어떨까..?
상하단 드래그와 아이템 드래그.
